Shell 重定向与管道

发表于 2018-01-10 | 分类于 Linux | 阅读次数
阅读时长 4

数据重定向

标准输入stdin：代码为0，使用<或<<
准输出stdout：代码为1，使用>或>>
标准错误输出stderr：代码为2，使用2>或2>>
>输出到一个已存在的文件中，那个文件就会被覆盖掉
>>输出到一个已存在的文件中，数据会在文件最下方累加进去

案例

find /home -name .bashrc 2> /dev/null stderr被丢弃
find /home -name .bashrc &> /dev/null 所有输出被丢弃
