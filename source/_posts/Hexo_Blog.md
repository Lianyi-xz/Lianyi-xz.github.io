### 前提条件
* 安装 [git](https://git-scm.com/downloads)
* 安装 [node.js](https://nodejs.org/en/)

### win10安装Hexo
```shell
#git bash
npm install hexo-cli -g
npm install hexo-generator-sitemap --save
npm install hexo-generator-baidu-sitemap --save
hexo -v
```
