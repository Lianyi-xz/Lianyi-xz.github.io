## 创建个人博客
### win 10安装Hexo
* 安装 [git](https://git-scm.com/downloads)
* 安装 [node.js](https://nodejs.org/en/)
* 安装`Hexo`  
```shell
#git bash
npm install hexo-cli -g
hexo -v
```
